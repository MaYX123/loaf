关于本站

本站是个人技术博客，主要发布原创博文。当前版本（2.y.z）为响应式布局，主要功能包括管理员撰写博文、后台管理、注册用户评论、markdown 编辑文章、语法高亮，推荐使用 Firefox、Chrome 浏览器。本站由 Bootstrap、jQuery、Font Awesome、Jinja2、Python 3.6、MySQL 支持搭建，部署在阿里云 ECS。源码可查看博主的 GitHub，欢迎你 star、fork、提出bug。

本站第一版源于廖雪峰老师的 Python 3 教程实战，由 Vue、jQuery、Jinja2、Python3.6、MySQL 搭建。博主目前更关注前端领域，但未来仍将使用 Django 重构后端。

2017年年末 @上海迪士尼
2017年夏 @南京江宁
关于博主

博主做过两年嵌入式开发，本站搭建之前对 web 开发一无所知，正是廖雪峰老师的 Python 实战，让我对 web 开发有了整体认识。第一版完成后，博主觉得前端太酷炫了，又能做设计又能写代码，于是在学了 JavaScript 后，立马计划用 React 重构前端，然而步子跨太大，就没有然后了。饭还是要一口一口吃，于是第二版用 Bootstrap 和 jQuery 重写了前端。第二版完成后才真正感受到前端领域之庞杂，仅仅用像 Bootstrap 这种傻瓜式框架是不行的。目前博主正徜徉在前端海洋中～
